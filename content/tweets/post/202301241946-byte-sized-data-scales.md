Numbers you should know:

5 bytes is ~1 token
1 web page is ~0.1MB
1 book is ~0.8MB
WikiText-103 is 0.5GB
BookCorpus is 5GB (GPT1)
WebText is 40GB (used for GPT2)
Pile is 800GB
BigPython is 200GB
Common Crawl is 0.4PT
Goog index is likely 8 PT
Humans see/hear ~1B tokens lifetime
